【作品簡介】
這是我的第一支由spring boot所完成的購物商城

主要的功能:

1.商品展示

2.購物車

3.後台管理

4.會員登入登出

【技術介紹】

網站開發:Spring boot+thymeleaf

安全認證:Spring Security

資料庫連接:spring data+mysql

前端優化功能:javascript+ajax

美化:bootstrap
